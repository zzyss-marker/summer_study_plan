### 1. 编码解码技术

#### 常见编码方式
```
1. Base编码系列:
   - Base64: 64个字符编码，常见于数据传输
     字符集: A-Z, a-z, 0-9, +, /
     填充字符: =
     特征: 长度为4的倍数，可能有1-2个=结尾

   - Base32: 32个字符编码
     字符集: A-Z, 2-7
     填充字符: =
     特征: 大写字母和数字2-7

   - Base16(Hex): 十六进制编码
     字符集: 0-9, A-F
     特征: 每个字节用两个十六进制字符表示

2. URL编码:
   - 格式: %XX (XX为十六进制ASCII码)
   - 常见字符: %20(空格), %3D(=), %26(&)
   - 双重编码: %2520 (编码后的%20)

3. HTML实体编码:
   - 命名实体: &lt; &gt; &amp; &quot;
   - 数字实体: &#60; &#62; &#38; &#34;
   - 十六进制: &#x3C; &#x3E; &#x26; &#x22;

4. Unicode编码:
   - UTF-8: 变长编码，1-4字节
   - UTF-16: 2或4字节编码
   - \u转义: \u0041 (A)
   - \x转义: \x41 (A)
```

#### 特殊编码识别
```
1. 摩斯密码:
   - 字符集: . - (点划)
   - 分隔: 字母间空格，单词间多个空格
   - 常见模式: .- (A), -... (B), -.-. (C)

2. 二进制编码:
   - 字符集: 0, 1
   - 长度: 通常8位为一组
   - ASCII对应: 01000001 (A)

3. 八进制编码:
   - 字符集: 0-7
   - 格式: \141 (a), 141 (a)

4. 培根密码:
   - 字符集: A, B 或 0, 1
   - 长度: 5位一组
   - 示例: AAAAA(A), AAAAB(B)

5. 栅栏密码:
   - 原理: 按栅栏排列后按行读取
   - 参数: 栅栏数(行数)
   - 破解: 尝试不同栅栏数

6. 凯撒密码:
   - 原理: 字母表位移
   - 参数: 位移量(0-25)
   - 破解: 暴力破解或频率分析
```

### 2. 隐写术 (Steganography)

#### 图片隐写

##### LSB隐写 (最低有效位)
```
原理:
- 修改像素值的最低位
- 人眼难以察觉微小变化
- 可隐藏大量数据

检测方法:
1. 统计分析:
   - 像素值分布异常
   - 相邻像素相关性降低

2. 视觉检测:
   - 提取LSB平面查看
   - 放大观察噪点

3. 工具检测:
   - stegsolve: 查看各个位平面
   - zsteg: 自动检测LSB隐写
   - steghide: 提取隐藏数据

提取方法:
- 按位提取LSB组成新图像
- 转换为二进制数据
- 解析隐藏的文件格式
```

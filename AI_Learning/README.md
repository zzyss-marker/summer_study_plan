# 🤖 AI机器学习与深度学习学习计划

## 🎯 学习目标
1. 掌握机器学习核心算法和数学原理
2. 熟练使用Python进行数据科学开发
3. 深入理解深度学习框架和神经网络
4. 完成3-5个实际项目，建立作品集

## 📚 学习路径

### 阶段一：Python基础强化 (第1周)
**目标**: 提升Python编程能力，为AI学习打基础

#### 核心内容
- Python高级特性 (装饰器、生成器、上下文管理器)
- NumPy数组操作和向量化计算
- Pandas数据处理和分析
- Matplotlib/Seaborn数据可视化
- Jupyter Notebook高效使用

#### 实践项目
- 数据清洗和探索性数据分析项目
- 股票数据分析可视化

### 阶段二：数学基础复习 (第1周)
**目标**: 巩固AI所需的数学基础

#### 核心内容
- 线性代数：矩阵运算、特征值、奇异值分解
- 概率统计：贝叶斯定理、概率分布、假设检验
- 微积分：偏导数、梯度、链式法则
- 最优化：梯度下降、拉格朗日乘数法

### 阶段三：机器学习基础 (第2周)
**目标**: 理解机器学习核心概念和算法

#### 核心算法
1. **监督学习**
   - 线性回归 (Linear Regression)
   - 逻辑回归 (Logistic Regression)
   - 决策树 (Decision Tree)
   - 随机森林 (Random Forest)
   - 支持向量机 (SVM)
   - K近邻 (KNN)

2. **无监督学习**
   - K-means聚类
   - 层次聚类
   - 主成分分析 (PCA)
   - t-SNE降维

3. **模型评估**
   - 交叉验证
   - 混淆矩阵
   - ROC曲线和AUC
   - 过拟合和欠拟合

#### 实践项目
- 房价预测 (回归问题)
- 鸢尾花分类 (分类问题)
- 客户聚类分析 (聚类问题)

### 阶段四：深度学习基础 (第3周)
**目标**: 理解神经网络原理和PyTorch框架

#### 核心内容
- 神经网络基础：感知机、多层感知机
- 反向传播算法原理
- 激活函数、损失函数、优化器
- 正则化技术：Dropout、Batch Normalization
- PyTorch基础：张量操作、自动求导、模型构建

#### 实践项目
- 手写数字识别 (MNIST)
- 从零实现神经网络

### 阶段五：深度学习进阶 (第4周)
**目标**: 掌握主流深度学习架构

#### 核心内容
1. **卷积神经网络 (CNN)**
   - 卷积层、池化层原理
   - 经典架构：LeNet、AlexNet、VGG、ResNet
   - 图像分类、目标检测基础

2. **循环神经网络 (RNN)**
   - LSTM、GRU原理
   - 序列建模和时间序列预测
   - 文本处理基础

#### 实践项目
- CIFAR-10图像分类
- 股票价格预测 (时间序列)

### 阶段六：计算机视觉项目 (第5周)
**目标**: 完成完整的计算机视觉项目

#### 项目选择 (选择1-2个)
1. **图像分类器**
   - 自定义数据集
   - 数据增强技术
   - 迁移学习

2. **目标检测**
   - YOLO或R-CNN实现
   - 标注工具使用

3. **图像生成**
   - 生成对抗网络 (GAN)
   - 变分自编码器 (VAE)

### 阶段七：自然语言处理项目 (第6周)
**目标**: 完成NLP相关项目

#### 项目选择 (选择1-2个)
1. **文本分类**
   - 情感分析
   - 新闻分类
   - Transformer模型使用

2. **文本生成**
   - 诗歌生成
   - 聊天机器人

3. **信息抽取**
   - 命名实体识别
   - 关系抽取

### 阶段八：项目优化和部署 (第7-8周)
**目标**: 优化模型性能，学习部署技术

#### 核心内容
- 模型压缩和量化
- 超参数调优
- 模型部署：Flask/FastAPI
- Docker容器化
- 云平台部署 (可选)
